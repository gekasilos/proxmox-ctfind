# proxmox-ctfind
Quick access to lxc containers in proxmox cluster

#### Install
Put script to bin directory which in your **$PATH**:
```console 
echo $PATH
```
```console
/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin
```
Example: 
```console
/usr/local/bin/ctfind
```

#### Usage
```console
ctfind name_or_partname
```
